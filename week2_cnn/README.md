# 第二周：MNIST 图像分类

本周使用 PyTorch 编写了一个小型卷积神经网络，对 MNIST 手写数字进行分类。

## 数据划分

- 训练集：20000 张图片
- 验证集：5000 张图片
- 测试集：10000 张图片

训练集用于更新模型参数，验证集用于观察训练效果和判断是否过拟合，测试集只在最后评价模型。

## 模型结构

模型包含两层卷积、两层池化和两层全连接层。训练时使用交叉熵损失函数和 SGD 优化器。

## 训练结果

模型训练 3 轮后，测试集准确率为 97.33%。

前两轮中，训练集和验证集的损失均明显下降。第三轮训练损失继续下降，但验证损失略有上升，说明模型出现了轻微过拟合。

## 输出文件

- `mnist_cnn.pth`：保存的模型参数
- `training_curves.png`：训练与验证曲线
- `confusion_matrix.png`：测试集混淆矩阵
- `error_samples.png`：部分错误分类样本
