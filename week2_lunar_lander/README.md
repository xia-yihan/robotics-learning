# LunarLander强化学习实验

这个项目使用 Gymnasium 的 LunarLander-v3 环境，比较了随机策略、人工启发式策略和DQN强化学习策略。

## 环境介绍

LunarLander的状态由8个数构成，包括位置、速度、角度、角速度以及两条腿是否接触地面。

智能体可以选择4种动作：

- 0：不启动发动机
- 1：启动左侧方向发动机
- 2：启动主发动机
- 3：启动右侧方向发动机

## 实验内容

### 随机策略

`lunar_lander_demo.py`随机选择动作。飞船没有根据状态进行判断，因此通常会坠毁。

### 启发式策略

`lunar_lander_heuristic.py`根据位置、速度和角度等状态，通过人工编写的规则选择动作，可以实现较稳定的着陆。

### DQN训练

`train_dqn.py`使用DQN进行初步训练。模型学会了避免快速坠毁，但测试时经常长时间悬停，没有稳定完成着陆。

`train_dqn_tuned.py`使用针对LunarLander调整后的参数重新训练。测试5轮时，飞船均成功着陆。

`test_dqn.py`用于加载训练后的模型并显示测试动画。

## 测试结果

调优后的DQN模型进行了5轮测试，结果如下：

| 测试轮次 | 总奖励 |
|---|---:|
| 1 | 295.39 |
| 2 | 152.94 |
| 3 | 282.26 |
| 4 | 278.11 |
| 5 | 287.20 |

5轮均成功着陆，平均奖励为259.18。不同回合的奖励存在差异，说明即使都能落地，着陆速度、姿态和发动机消耗等过程也会影响最终得分。

## 主要收获

通过这个实验，我理解了强化学习中的状态、动作、奖励和策略，也看到了三种决策方法的区别：

- 随机策略没有明确目标
- 启发式策略依赖人工规则
- DQN通过与环境交互和奖励反馈自动学习策略

实验也说明，网络结构、探索比例、学习率和更新频率等参数会明显影响训练结果。
